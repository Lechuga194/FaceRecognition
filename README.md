# FaceRecognition
Reconocimiento de rostros mediante la API de clarifai.

Construida con React.js y Node.js
Prueba la app aqui: https://lechuga194.github.io/FaceRecognition/

![enter image description here](https://i.ibb.co/WPrm6hD/a.png)

Para usar la app localmente deberas seguir los siguentes pasos

 1. Necesitaras una KEY para la API usada, puedes conseguirla aqui https://www.clarifai.com/
 2. Clona el repositorio
 3. Crea una base de datos con el nombre 'facerecognitionapp'
 4. Crea la tabla 'users' con el archivo db.psql que esta dentro de la carpeta database
 5. Modifica el archivo .env dentro de la carpeta server con los respectivos datos
 6. Dentro de la carpeta server ejecuta 'npm install' y posteriormente 'npm start' esto instalara e iniciara el servidor (El puerto por defecto es 3000, lo puedes cambiar en el archivo .env)
 7. En el directorio raiz ejecuta nuevamente 'npm install' y posteriormente 'npm start' esto abrira una pestaña en el navegador predeterminado con el frontend de la aplicacion
 8. Listo!
